@ Definition  of the database  system
The term ‘DBS’ refers to a computer-centered  system that creates necessary  information  by storing and managing
data in a database.

@ Components  of the database  system
As regards  the processing  of data stored in a database,  there is a user who processes  it, a language  that can
manipulate  and read data in the database,  and DBMS software  that enables  the processing  of all data.
That is, a database  system is composed  of four attributes:  database,  database  language,  user, and database
management  system (DBMS).
