05 Routing  Protocol  Algorithm  and Type
A) What is a routing  protocol?
Routing  protocol  defines  the types of messages  exchanged  between  routers,  exchange  procedure,  and actions  when
receiving  a message,  for efficient  setting  and updating  of the routing  tables.
B) What is a routing  algorithm?
The routing algorithm  finds the low-cost  path between  a source router and a destination  router in a graphical
network,  connected  by links with a cost between  routers. The low-cost  path refers to the path of minimum  value
among  the sum of link costs out of all possible  paths from the source to the destination.  <Table 49> shows the types
of routing  algorithms.
