When classifying routing protocols,  based on AS, the routing protocol  within the AS is called the inter gateway
protocol  (IGP). The leading IGP includes  the routing information  protocol  (RIP), using a distance  vector routing
algorithm,  and the open shortest  path first (OSPF),  using a link-state  routing  algorithm.  The routing  protocol  between
AS is called the exterior  gateway  protocol  (EGP). The leading  EGP is the border  gateway  protocol  (BGP) that uses a
path-vector  routing  algorithm.
