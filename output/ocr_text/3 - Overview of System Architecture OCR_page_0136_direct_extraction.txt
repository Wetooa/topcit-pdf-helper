storage,  and network,  which are independently  operated  between  containers.

Container  virtualization  technology  can be characterized  as being lighter and more portable  than existing
virtualization  technologies.  In Linux containers  (LXC), there is kernel space for managing  physical resources,  and a
user space for executing  user processes  inside the OS. The container  virtualization  technology  partitions  the user
space into ten and limits the resources  visible in each user process.

OS-level  virtualization  (container  virtualization)  has the following  advantages:
* Quick start and finish: Very fast compared  to hypervisor-based  virtualization
+ High density:  Minimized  resource  consumption,  since an OS can operate  multiple  containers
+ Low overhead:  Userspace  isolated  without  emulation
