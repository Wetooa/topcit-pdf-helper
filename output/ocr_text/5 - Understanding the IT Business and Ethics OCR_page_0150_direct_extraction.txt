+ Server  virtualization
By consolidating  dozens of physical server workloads  within the data center on to several virtual servers,  the
utilization  of server resources  can be improved  (physical  rack costs, administrative  costs, and green IT).
+ Storage  virtualization
In place of the physical storage space required,  a technology  called Thin-Provisioning  virtually  allocates  only
the initial minimum  required  space to enable service implementation.  It also provides  an environment  where
heterogeneous  storage  systems  can be integrated  and used.
+ Network  virtualization
L2, L3, L7 switches,  network  firewalls,  and security  equipment  that had existed in the form of hardware  appliances
are implemented  as virtual machines.  This allows network  resources  to operate internally  through  virtualization,
even ina single shared physical  environment.
