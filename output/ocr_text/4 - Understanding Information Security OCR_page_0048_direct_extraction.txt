Understanding  the latest information  security  threats  and response  technologies
New types of cyber-attacks  emerge,  such as the combination  of cyber-attacks  and social engineering
techniques,  and cyber convergence  attacks, by combining  various cyber-attack  technologies,  etc.
Accordingly,  the security  threats of the financial  IT environment  are gradually  becoming  more
intelligent,  advanced,  and diversified.  As loT and big data emerge as core technology  that can
influence  the competitiveness  of companies  and countries  in the future,a specific plan for applying
security  is urgently  requiredto  fully promote  those technologies.
>>b> Learning  objectives
To be able to explain the latest information  security  threats  and countermeasures.
To be able to explain information  security  technology  for responding  to security  threats in a cloud
environment.
To be able to explain information  security  technology  for responding  to security  threats in a big data
environment.
To be able to explain information  security  technology  for responding  to security  threats in a web/
mobile environment.
