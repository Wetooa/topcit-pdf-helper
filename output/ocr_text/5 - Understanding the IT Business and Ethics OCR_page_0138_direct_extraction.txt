In addition  to the responsive  web, adaptive  web technology  is attracting  a lot of interest  as a technology  that enables
websites  to provide a seamless  transition  of information  for different  screen sizes. The adaptive  web is a server-
based or client-based  technology  that checks  the device that is trying to access the web. By using images tailored
for the device  of the user, the adaptive  web allows website  access  on a mobile device at high speeds.

Web that enables  intelligent  data processing  and the accumulation  of knowledge  through  the defining  and building  of
structured  data (ontology)  and data linking among  the structured  data is called semantic  web.
The web, which only displays  web documents  in HTML language,  has severe limitations  in expressing  resources,  and
the relationship  between  web documents  is defined. Hence computers  cannot understand  semantic.  Since there
are problems  in recognizing  all objects  with just URL alone, semantic  web was developed  as a way to efficiently  use
information  from distributed  webs, extract  meaningful  data from flood of information, and as a means for structuring
and interpreting  the huge amounts  of data produced  on the web.
