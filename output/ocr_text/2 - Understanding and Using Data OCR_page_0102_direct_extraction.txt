01 Understanding  the Concept  of Relational  Algebra
Relational  algebra is largely divided into relational  algebra and relational  calculus.  Unlike relational  analysis,  which is a
non-procedural  language  that defines  desired information, a relation  operation  has the characteristic  of a procedural
language  because  it specifies  the order of operations  to be performed in order to obtain desired information.  In
particular,  as relational  algebra becomes  the theoretical  basis for SQL, the standard  query language  of relational
databases,  the concept  and principle  of relational  algebra need to be clearly understood.  Therefore,  this module
introduces  set operation  and relation  operation,  which constitute  the basic relational  algebra, and the extended
relational  algebra  operation.
O2 Set Operation  and Relation  Operation

operation,  and the result of the operation  is relations.  Basic relational  algebra operations  are usually divided into set
operations  and relation  operations.
