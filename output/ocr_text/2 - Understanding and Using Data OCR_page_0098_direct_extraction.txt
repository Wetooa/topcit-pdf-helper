02 Data Standardization
Data standardization  means  that principles  for the name, definition,  format, and rule of data information  elements  are
established  and applied throughout  an organization.  It is indispensable  for improving  the clarity of communication  and
data quality and for reducing  the cost of data conversion  and cleansing.
A) Overview  of data standardization
@ Recently, interest  in data integration  and data quality has been increasing  because  data have emerged  as a key
element  of a corporation's  strategic  decision-making  process.
@ Data standardization  means that the principles  for the name, definition,  format,  and rule of data information  elements,
which are dispersed  in each system,  are established  and applied throughout  an organization.
@ The exact meaning  of data can be  understood,  and opposite  opinions  about data can be adjusted  through data
standardization.
B) Necessity  of data standardization
@ Improves  clarity of communication  by unifying  names.
@ Reduces  the time and effort spent on finding  the location  of the necessary  data.

@ Reduces  the cost of data conversion  and cleansing  when designing  a data interface  between  information  systems.
